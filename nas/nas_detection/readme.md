### 1.文件结构

- nas_det.py - 主程序，用于展示可视化ui

- nas_model_vis.py - NAS搜索过程中模型变化的可视化程序

- figure - 保存的搜索模型图片

### 2. 环境配置

 - python3.5以上版本

 - qtcreator，用于编辑qt的ui文件
 
 - pyqt5 统一使用5.12版本，安装如下

    ```shell
    pip install pyqt5==5.12
    pip install PyQtWebEngine==5.12	
    ```
 ### 3. 运行	

```shell
	python nas_det.py
```
